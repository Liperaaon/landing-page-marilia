Landing page institucional desenvolvida sob medida para a especialista em Psicopedagogia e Neuropsicopedagogia, **Marília**. O projeto foi concebido para ser o principal canal de captação de pacientes, com um design focado em transmitir acolhimento, confiança e autoridade profissional para pais que procuram ajuda especializada para os seus filhos.

## 🚀 Contexto do Projeto

Fui contratado para digitalizar a presença da profissional, criando uma plataforma que fosse além de um simples cartão de visita digital. O desafio foi transformar a sensibilidade do atendimento presencial numa experiência de navegação acolhedora, orientando os pais de forma clara através dos serviços de avaliação, planeamento e acompanhamento psicopedagógico.

### ✨ Diferenciais Técnicos e Funcionalidades

* **Design Emocional e Acolhedor:** Utilização de uma paleta de cores quentes (tons terrosos e pastéis) combinada com tipografias elegantes (*Lora* e *Inter*) e efeitos modernos de *glassmorphism* para criar um ambiente digital que transmite calma e segurança.
* **Performance e Zero-Dependency UI:** Construído de forma nativa e otimizada (HTML5 e Vanilla JS), utilizando o Tailwind CSS via CDN com configurações injetadas diretamente, garantindo um carregamento ultrarrápido sem a necessidade de frameworks pesados.
* **Animações Dinâmicas (Scroll Reveal):** Implementação da `Intersection Observer API` em JavaScript puro para criar animações suaves (reveal, pulse e blobs) que guiam a leitura do utilizador à medida que este faz scroll na página.
* **Foco na Conversão (UX/UI):** Estrutura orientada para a captação direta de *leads*, com botões flutuantes e *Call-to-Actions* (CTAs) estratégicos integrados com o WhatsApp para agendamentos imediatos.
* **Design Responsivo e Mobile-First:** Experiência de navegação perfeita em qualquer ecrã, com menus *off-canvas* otimizados para smartphones, dado que a maioria do público-alvo (pais) acede através de dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5 semântico, Tailwind CSS customizado (injetado via script de configuração).
* **Interatividade:** Vanilla JavaScript (ES6+) para gestão de estado do menu mobile, cálculos de ano dinâmico e observação de intersecções no DOM.
* **Integrações e Assets:** FontAwesome para iconografia vetorizada, Google Fonts e meta tags otimizadas para partilha social (OpenGraph).

## 📱 Visibilidade e Impacto

O resultado é uma aplicação leve, responsiva e de alta conversão. A arquitetura da página não só melhora o posicionamento orgânico como atua como um "concierge" digital, que acolhe a dor dos pais (dificuldades escolares dos filhos) e apresenta a metodologia de intervenção de forma empática e transparente, reduzindo a fricção no contacto inicial.
